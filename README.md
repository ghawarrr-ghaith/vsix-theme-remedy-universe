# Remedy Universe Themes

A collection of VS Code themes inspired by games from Remedy Entertainment, including Control and Alan Wake.

## Available Themes

*   **Alan Wake: The Writer** - Forest greens and flashlight beams. Inspired by the Pacific Northwest atmosphere.
*   **Alan Wake: Dark Place** - Oppressive blacks, neon pinks, and spirals. Captures the nightmare reality of the Dark Place.
*   **Control (Jesse Faden)** - Deep, dark backgrounds (`#1c1c1c`) with prominent "Hiss Red" accents. Based on the brutalist architecture of the Oldest House.
*   **Control (Astral Plane)** - A blindingly stark Light Theme (`#f4f4f4`) with void-black text, mimicking the Astral Plane.
*   **Control (Black Rock)** - An ultra-dark theme (`#0a0a0a`) based on Black Rock prisms, with gold/yellow accents.

## Installation

1.  Open Chrome or your browser of choice.
2.  Download the `.vsix` file.
3.  Open VS Code.
4.  Go to Extensions -> "..." -> Install from VSIX...
5.  Select the downloaded file.
6.  Go to File -> Preferences -> Color Theme and select one of the Remedy Universe themes.
