# Remedy Universe Themes

A collection of Visual Studio Code themes inspired by the games developed by **Remedy Entertainment**. This extension brings together themes from *Control* and *Alan Wake*.

## Included Themes

### Alan Wake
- **The Writer:** Forest greens and flashlight beams. Inspired by the Pacific Northwest atmosphere.
- **Dark Place:** Oppressive blacks, neon pinks, and spirals. Captures the nightmare reality of the Dark Place.

### Control
- **Control (Jesse Faden):** Deep, dark backgrounds (`#1c1c1c`) with prominent "Hiss Red" accents. Based on the brutalist architecture of the Oldest House.
- **Control (Astral Plane):** A blindingly stark Light Theme (`#f4f4f4`) with void-black text, mimicking the Astral Plane.
- **Control (Black Rock):** An ultra-dark theme (`#0a0a0a`) based on Black Rock prisms, with gold/yellow accents.

## Installation

1. Open **Extensions** sidebar in VS Code.
2. Search for `Remedy Universe Themes`.
3. Click **Install**.
4. Go to **File > Preferences > Color Theme** and select your preferred Remedy theme.
